Introduction

Use the template below to write a comment for the scoping phase of Seattle's 2024 Comprehensive Plan Update. Fill in the blanks with options we've provided or come up with your own!

In this phase, the city decides the range of land use possibilities it should study and eventually implement. The city has proposed 5 alternatives ranging from no change (Alternative 1) to allowing a bit more multi-family housing in some areas (Alternatives 2-5).

We believe that none of these alternatives go far enough and thus call for an Alternative 6. Join us!

If you want to learn more about Seattle's Comprehensive Plan Update process, check out this flyer and other additional resources.

Step 2 of 2: Submit Comment

Make any edits you like to the text within the box above. When you are done, go to One Seattle's comment portal and click "submit a comment" on the right hand side.

If you don't have an account yet, you'll need to create one before proceeding (see note below). If you already have one, you'll need to log in.

Copy-paste your comment into the description box. Start the title of the comment with “Alternative 6” and then add whatever else you want.

If you don't want to create an account: We encourage you to create an account so that you can upload your inspirational density day dreams to the official hub for us to upvote. You can, however, also email your comment directly to the Seattle planning department instead: brennon.staley@seattle.gov.
